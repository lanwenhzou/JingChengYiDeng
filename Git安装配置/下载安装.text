一：下载：https://msysgit.github.io
二.安装：
    1.选项：Adjusting your path => 批处理模式 use gitfrom Git bash only 选中
三 配置PATH ：根据安装目录路径配置
四 配置 Git：打开终端 => git bash (windows) ， 检测git 版本 git --version (Linux)
            1. 打开配置文件查看 git config --global -e   //此处是配置全局的GIT 文件
            2. 配置用户名和邮箱：
                1. git config --global user.name "用户名"
                2. git config --global user.email "邮箱"
